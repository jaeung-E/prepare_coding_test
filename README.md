# 브랜치 네이밍

**연-월-일/이름** 순으로 적용

`ex) 2022-08-30/jaeung`

<br>

# 폴더 구조

**플랫폼/난이도/문제_이름/이름.js** 구조로 작성

```
ex)
- programmers/level_1/모음_사전/jaeung.js
- boj/silver/숨바꼭질/jaeung.js
```

<br>

# 코드 리뷰

- `main` 브랜치를 기준으로 Pull-Request 생성

- 생성된 `PR`을 기준으로 궁금한 점, 고치면 좋을 것 등 자유롭게 의견 남기기

- 별도의 의견이 없을 경우, LGTM 가능
